## Задание 3.

Информация о багаже пассажира описывается массивом, где каждый элемент содержит название
единицы багажа (чемодан, сумка, коробка и т.д.) и ее массу. Дан файл, содержащий сведения о
багаже нескольких пассажиров. Найти багаж, средняя масса одной единицы багажа, в котором
отличается не более чем на m кг от общей средней массы одной единицы багажа.

## Алгоритм решения:

Пользователь вводит число m. Создается массив initialData с названиями предметов и нулевым весом у каждого. Далее пользователь вводит вес каджого предмета. Затем с мощью метода fill(в который передается название файла и initialData) заполняем бинарный файл данными. Затем читаем данные из файла методом readAll и записываем в allBaggage. Создаем переменную double totalWeight = 0; в которой будет записан общий вес багажа. Проходим по всем элементам allBaggage в цикле for и плюсуем к totalWeight. Получаем среднее арифметическое double average = totalWeight / allBaggage.size();. Проходим циклом по списку и для каждого предмета проверяет формулу: std::abs(item.weight - average) <= m — если эта разница не превышает число m, багаж считается подходящим.
