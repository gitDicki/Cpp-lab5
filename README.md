## Задание 1.

Получить в новом файле те компоненты исходного файла, которые являются четными.

## Алгоритм решения:

Пользователь вводит количество чисел, которые запишутся в файл. Затем с помощью метода fillRandom в бинарынй файл записываются числа. Затем читаем данные из файла 1 и записываем в файл 2 только четные (используем остаток от деления для  нахождения четных),с помощью метода readAllNumbers читаем данные, которые записались во второй файл и выводим в терминал.

## Задание 2.

Скопировать элементы заданного файла в квадратную матрицу размером n×n (если элементов
файла недостает, заполнить оставшиеся элементы матрицы нулями). Поменять местами
в каждой строке минимальный и максимальный элементы.

## Алгоритм решения:

Пользователь вводит количество чисел, которые запишутся в файл. Затем с помощью метода fillRandom в бинарынй файл записываются числа. Далее создается матрица из чисел, которые в файле с помощью метода createMatrixFromFile. Пользователь вводит размер квадртаной матрицы и она заполняется нулями, затем читаются числа из файла и записываются в массив numbers. Используем два цикла for, чтобы пройти по всех столбцам всех строк. Затем в циксле for сравнивается каждый элемент строки с текущим найденным минимумом/максимумом.После того как значения найдены, вызывается функция swapIter, которая ищет в строке row первое вхождение значения minElem (и max) с помощью std::find меняет их местами.
